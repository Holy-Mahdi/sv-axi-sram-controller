# SV AXI SRAM Controller

A SystemVerilog implementation of an AXI4-Lite to SRAM controller.

## Project Structure

- `rtl/`: Contains the design files (Interface, Controller, SRAM Model).
- `tb/`: Contains the testbench files.
- `sim/`: Simulation scripts.
